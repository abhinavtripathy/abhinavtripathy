## About Me

I'm a software engineer based in Arlington, VA, currently working at **Amazon Web Services (AWS)**. I studied Computer Science at UMass Amherst where I got deep into open source, hackathons, and building things from scratch.

I enjoy working across the stack — from cloud infrastructure to frontend UIs — and I've contributed to projects ranging from a core Ansible bug fix to building a programming language designed to teach beginners.

- Building distributed systems and cloud infrastructure at **AWS**
- Contributed to **Ansible**, **HackUMass (fuseumass)**, **DSC UMass**, and more
- Built [Pharos](https://github.com/abhinavtripathy/pharos), a beginner-friendly programming language where you write code in plain English
- Co-organized **Summer with DSC UMass** to help students break into open source

## Open Source Contributions

### [ansible/ansible](https://github.com/ansible/ansible)

Fixed a bug preventing Module Logging from working correctly in WSL2 environments — traced the issue to how Ansible resolved log paths under the Windows Subsystem for Linux and patched it to handle the WSL2 path correctly. [[PR #76375]](https://github.com/ansible/ansible/pull/76375) ✓ merged

### [fuseumass/dashboard](https://github.com/fuseumass/dashboard) — HackUMass Organizer Platform

Core contributor to the open-source hackathon management platform powering [HackUMass](https://hackumass.com), one of the largest hackathons in the Northeast. Shipped 15+ merged PRs including:

- Email verification feature end-to-end (DB migration → backend → frontend)
- Mobile-optimized judging UI with two-column layout and project detail modals
- CSV export/import for event applications, hardware inventory, and event schedules
- Hardware location tracking with Rails database migrations
- Expanded college autocomplete to 3,000+ US & Canadian institutions

### [dsc-umass](https://github.com/dsc-umass)

Contributed to **social-insights** (refactoring, test coverage) and **project-matching** (Docker Compose setup) for the Developer Student Club at UMass Amherst.
