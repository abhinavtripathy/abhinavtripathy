## About Me

I'm a software engineer based in Arlington, VA, currently working at **Amazon Web Services (AWS)**. I studied Computer Science at UMass Amherst where I got deep into open source, hackathons, and building things from scratch.

I enjoy working across the stack — from cloud infrastructure to frontend UIs. On the open source side, I've contributed to [Ansible](https://github.com/ansible/ansible) (the widely-used IT automation and configuration management tool) with a bug fix for Module Logging in WSL2 environments, and to [FuseUMass](https://github.com/fuseumass/dashboard) (the open-source hackathon organizer platform that powers HackUMass, one of the largest college hackathons in the Northeast) where I shipped 15+ merged PRs covering email verification, judging UI, CSV pipelines, and hardware tracking.

- Built [Pharos](https://github.com/abhinavtripathy/pharos), a beginner-friendly programming language where you write code in plain English
- Contributed to [DSC UMass](https://github.com/dsc-umass) — the Developer Student Club at UMass Amherst
- Co-organized **Summer with DSC UMass** to help students break into open source
